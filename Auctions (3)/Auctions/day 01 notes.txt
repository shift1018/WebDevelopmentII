NOTE 1:

Your system probably has ability to execute PS1 scripts disabled by default.
It has to enabled globally for some of the tools to work.

To fix this Start PowerShell as Administrator, then execute this command, answer the question "A" for All

Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy Unrestricted
